VGP (status: em andamento):
    Versão do VideoGameEngSoft. utilizando o framework Spring, trata-se de um sistema de gerenciamento para uma microempresa de manutenção de viedogames.
    -foi feito apenas o backend da aplicação utilizando as dependencias: jpa, web, devtools e mysql connector.
    -futuramente pretendo adicionar mais funções e adicionar o front end..
Pre-requisitos:
    -Java 11
    -banco de dados MySql
Guia de instalação:
    -criar conexão no banco de dados na porta 3306 com login: 'root' sem senha.
    -abrir o projeto e executar
